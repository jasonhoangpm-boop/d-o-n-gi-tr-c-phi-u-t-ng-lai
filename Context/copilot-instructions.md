# Copilot Instructions

This project predicts stock prices using deep learning.

## Rules

- Use Python
- Use PyTorch
- Write modular code

- Normalize data
- Use MinMaxScaler
- Create sliding window sequences

## Training

- Use Adam optimizer
- Save best model
- Print training loss
